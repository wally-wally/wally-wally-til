# 03-13. 반복문


# (1) 반복문을 파이프라인으로 바꾸기

  • 컬렉션 파이프라인을 이용하여 기존에 사용했던 반복문을 일련의 연산으로 간단하게 표현할 수 있다.
    • map : 입력 컬렉션의 각 원소를 반환
    • filter : 또 다른 함수를 사용해 입력 컬렉션을 필터링해 부분집합을 만듬
  • 논리를 파이프라인으로 표현하면 이해하기 훨씬 쉬워진다.
    • 객체가 파이프라인을 따라 흐르며 어떻게 처리되는지를 읽을 수 있기 때문이다.
